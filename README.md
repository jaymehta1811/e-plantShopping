Paradise Nursery is a React-based plant shop application that allows users
to browse houseplants by category, add them to a shopping cart, and manage
quantities using Redux state management.

## Features
- Categorized plant listings
- Add to Cart functionality
- Redux-based shopping cart
- Quantity management
- Dynamic total cost calculation

## Technologies Used
- React
- Redux Toolkit
- React Router
- CSS
